My current work-owned machine is a Surface Pro 4. It's a thin and light device that can be configured with up to 16GB of RAM.

> "There's no way I can be productive on a tablet!"

A tablet is definitely smaller than a 30" monitor. But, do you *really* need all the pixels? I'm not currently using any external screens with the Surface Pro 4. And, that's totally OK!

## UI Clutter / Waste

Many applications waste a good amount of screen space. Modern operating systems can too.

I'm talking about persistently visible UI elements. Taskbars, toolbars, menu bars, ribbons, etc. **As the screen shrinks, the percentage of wasted space increases.**

If you're feeling cramped **cut the clutter**! In text editors it might mean turning off some visual elements. Full screen can be helpful, too! In Windows, give <kbd>F11</kbd> in various apps a try!

**Spending a few minutes here can lead to a beautifully distilled development environment.**

## Touching the screen

If you've got a tablet, you'll probably want to touch the screen. **That means fingerprints.**

**Bright backgrounds help hide fingerprints.** Configure your command line, editors, etc. to use a light theme *(as in, a light background)*. Dark backgrounds may give you *cool points*, but they don't help you see through fingerprints!

Plus, you'll be all set for using a projector to show off your code, as light backgrounds and dark text are generally superior for that use case.

## Focus

Having less screen clutter can put your mind at ease, similar to having a clean and uncluttered home. In fact, you might even enjoy the clarity that comes with seeing *only what matters*.

Now that you're calm, relaxed, and focused you may find you produce better work as a result.

## Flexibility

Not having an *always there* built-in keyboard is an opportunity. An impromptu pairing session with a wireless keyboard means the tablet can sit between you, rather than in front of the current keyboard user.

**Want a portrait screen? Rotate it. Want to float the tablet above your desk? Buy a clamp.**

Plus, it's easier to carry around. Using a smaller bag, rather than a backpack or messenger bag is a nice added benefit.

## It's a great way to work!

This might not be for everyone. I'm not currently into game development, and this setup might be ill-suited for that. But, having a thin, light, and usable-during-takeoff-and-landing machine might be just what you're looking for.

**Embrace the small screen!**
